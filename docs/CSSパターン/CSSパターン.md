# CSS パターン集

ConiferFruits で使える 5 つのデザインスタイル見本。各ページに同名の .css が紐付いて自動適用される。

## 一覧

- [1. コーポレート](./1-コーポレート.md) — ネイビー縦バー＋グラデーション、業務文書向け
- [2. 下線アクセント](./2-下線アクセント.md) — 蛍光マーカー風グラデーション下線
- [3. 左バー](./3-左バー.md) — 緑の太い L 字型縦バーで階層を表現
- [4. 連番チップ](./4-連番チップ.md) — 自動連番を丸チップ形式で付与
- [5. 背景塗り](./5-背景塗り.md) — 見出しを黒・濃グレー背景で塗りつぶし

## カスタマイズ

各 .css をコピーして自分のフォルダに置けばそのフォルダ配下に適用される。
編集も可能。
